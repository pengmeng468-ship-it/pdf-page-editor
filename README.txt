PDF Editor-lite

使用方法：
1. 双击 PDF工具.exe。
2. 浏览器自动打开后，点击“添加 PDF”选择一个或多个 PDF。
3. 选择页面后，可左转、右转、删除，也可拖拽缩略图调整顺序。
4. 需要按范围拆分时，在输入框填写 1-3,8,10-12 这类范围，再点击“拆分”。
5. 点击“导出 PDF”会弹出保存窗口；如选择已有文件，系统会询问是否覆盖。

Word/PPT 转 PDF：
1. Word 文件：点击“Word转PDF并编辑”或“Word转PDF保存”。
2. PPT 文件：点击“PPT转PDF并编辑”后会先作为普通 PDF 加入编辑区。
3. PPT 导入完成后，可选择“不合并 / 2合1 / 4合1 / 6合1 / 8合1”，也可选择“从左到右 / 从上到下”；每次调整都会自动刷新缩略图预览。
4. “PPT转PDF保存”会按当前选择的合并方式和排列方向直接保存 PDF。
5. PPT 多页合一输出固定为 A4 竖版。

运行要求：
- Windows 10 / Windows 11。
- 普通 PDF 编辑不需要联网。
- Word/PPT 转 PDF 需要安装 Microsoft Office 或 LibreOffice。
- 程序会优先使用 Microsoft Office，失败时尝试 LibreOffice。

说明：
- 程序只做页面级处理，不直接修改 PDF 原有文字。
- 原文件不会被自动覆盖；导出和拆分都会生成新文件。
- 支持中文路径、中文文件名、带空格的路径。
- 如遇到损坏或加密 PDF，界面会显示错误。
- 错误日志在：%LOCALAPPDATA%\PDF Editor-lite\error.log
