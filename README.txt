PDF Editor-lite

这是一个 Windows 本地 PDF 页面编辑工具。双击 PDF工具.exe 后，会自动打开浏览器界面。

常用功能：
1. 添加 PDF 后，可拖拽页面缩略图调整顺序。
2. 选中页面后，可左转、右转、删除。
3. 输入 1-3,8,10-12 这类范围，可拆分 PDF。
4. 点击“导出 PDF”保存新文件。
5. 左侧文件列表可选中或移除整个导入文件。

Word 转 PDF：
- “Word转PDF并编辑”：转换后加入编辑区。
- “Word转PDF保存”：只转换并保存。

PPT 转 PDF：
- “PPT转PDF并编辑”：先作为普通 PDF 加入编辑区。
- 导入多个 PPT 时，先在左侧点选要调整的 PPT。
- 导入后选择“不合并 / 2合1 / 4合1 / 6合1 / 8合1”。
- 可选择“从左到右 / 从上到下”排列。
- 每次调整合并方式或排列方向，只刷新当前选中的 PPT。
- “PPT转PDF保存”会按当前设置直接保存 PDF。

运行要求：
- Windows 10 / Windows 11。
- 普通 PDF 编辑不需要联网。
- Word/PPT 转 PDF 需要安装 Microsoft Office 或 LibreOffice。

说明：
- 程序只做页面级处理，不直接修改 PDF 原有文字。
- 原文件不会被自动覆盖。
- 支持中文路径、中文文件名和带空格路径。
- 错误日志在：%LOCALAPPDATA%\PDF Editor-lite\error.log
